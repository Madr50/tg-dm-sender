# Telegram DM Sender - Flask Web Version

## الوصف

نسخة ويب من أداة إرسال روابط الدعوة بتيليجرام. تشتغل بشكل مستمر على السيرفر مع لوحة تحكم ويب كاملة، ومتوافقة مع Uptime Robot.

---

## المميزات

| الميزة | الوصف |
|--------|-------|
| **Dashboard ويب** | لوحة تحكم كاملة للتحكم بالأداة |
| **Uptime Robot** | endpoint `/health` متوافق مع Uptime Robot |
| **Background Running** | الأداة تشتغل بالـ background ما تنطفي |
| **Start/Stop/Pause** | تحكم كامل عبر الواجهة |
| **Stats Live** | إحصائيات حية تتحدث كل 5 ثواني |
| **Activity Log** | سجل نشاط مباشر |
| **Persistent State** | الحالة تتحفظ حتى لو أعيد تشغيل السيرفر |
| **Anti-Ban** | نفس نظام تجنب الحظر المتقدم |

---

## طريقة التشغيل

### 1. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 2. تشغيل الأداة

```bash
python app.py
```

### 3. فتح اللوحة

افتح في المتصفح: `http://YOUR_SERVER_IP:5000`

### 4. إعداد Uptime Robot

أضف الرابط التالي كـ HTTP(s) Monitor:
```
http://YOUR_SERVER_IP:5000/health
```

---

## روابط API (لـ Uptime Robot)

| الر endpoint | الوصف |
|-------------|-------|
| `/` | لوحة التحكم الرئيسية |
| `/health` | فحص الصحة (Uptime Robot) |
| `/api/stats` | الإحصائيات الحالية (JSON) |
| `/api/logs` | آخر 50 سجل نشاط (JSON) |
| `/api/start` | بدء الحملة (POST) |
| `/api/stop` | إيقاف الحملة (POST) |
| `/api/pause` | إيقاف مؤقت / استئناف (POST) |
| `/api/reset` | إعادة تعيين الإحصائيات (POST) |

---

## طريقة الاستخدام

1. شغّل الأداة على السيرفر
2. افتح اللوحة بالمتصفح
3. أدخل رابط الدعوة ونص الرسالة
4. اضغط **Start**
5. الأداة تشتغل بالـ background وما تنطفي

---

## متغيرات البيئة

| المتغير | القيمة الافتراضية | الوصف |
|---------|-------------------|-------|
| `PORT` | 5000 | البورت اللي يشتغل عليه السيرفر |

## ملاحظة مهمة

- لا تستخدم حسابك الرئيسي
- الأداة تتوقف تلقائياً بعد 3 أخطاء Flood متتالية
- كل ما تحتاجه هو أن السيرفر يشتغل 24/7 والـ Uptime Robot يضمن ذلك
